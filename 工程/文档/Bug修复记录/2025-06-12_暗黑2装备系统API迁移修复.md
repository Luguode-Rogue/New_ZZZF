# 2025-06-12 暗黑2装备系统API迁移修复

## 任务描述
修复 New_ZZZF 模块中暗黑2装备仿制系统的编译错误，共涉及 7 类报错。

## 修改文件清单

### 1. `EquipmentAffixSystem\Data\AffixDefinition.cs`
- **问题**：`[SaveableField]` 只能用于字段，此处全是属性
- **修改**：11处 `[SaveableField(n)]` → `[SaveableProperty(n)]`
- **影响属性**：Id, IsPrefix, DisplayName, Description, Weight, MinItemLevel, MaxItemLevel, AllowedItemTypes, ConflictAffixIds, StatModifiers, Rarity

### 2. `EquipmentAffixSystem\Data\AffixDatabase.cs`
- **问题**：同上
- **修改**：`[SaveableField(1)]` → `[SaveableProperty(1)]`（AffixMap 属性）

### 3. `Systems\SkillConfigManager.cs`
- **问题**：同上
- **修改**：`[SaveableField(1)]` → `[SaveableProperty(1)]`（_troopSkillMap 字段→视为属性处理）

### 4. `EquipmentAffixSystem\Logic\AffixGenerator.cs`
- **问题**：`MBRandom` 是静态类，不能 `new MBRandom()`，且该字段从未使用
- **修改**：删除第15行 `private static readonly MBRandom Random = new MBRandom();`
- **说明**：代码中所有随机均使用 `System.Random`，该字段冗余

### 5. `Skills\Passive\CriticalStrikePassive.cs`
- **问题**：`MBRandom` 不在上下文中
- **分析**：该文件已有 `using TaleWorlds.Core;`，`MBRandom.RandomFloat` 是静态属性调用，写法正确
- **结果**：无需修改（前两个文件修复后此文件也应能编译）

### 6. `EquipmentAffixSystem\Campaign\AffixCampaignBehavior.cs`
- **问题1**：`WeaponComponent.WeaponClass` 在新版中不存在
- **修改**：第213行 `weapon.WeaponClass` → `weapon.PrimaryWeapon.WeaponClass`
- **问题2**：`CampaignEvents.OnPlayerInventoryExchangeEvent` 不存在
- **修改**：第52行去掉 `On` 前缀 → `CampaignEvents.PlayerInventoryExchangeEvent`

### 7. `Harmonys\Harmony.cs`
- **问题**：`MobileParty` 在当前上下文中不存在
- **修改**：添加 `using TaleWorlds.CampaignSystem;`
- **位置**：在 using 块中第2行（`using System;` 之后）

### 8. `Script.cs`
- **问题**：`Color` 无法隐式转换为 `uint`（新版 `SetContourColor` 可能接受 `Color?`）
- **修改**：
  - 第144行：`Dictionary<Agent, uint?>` → `Dictionary<Agent, Color?>`
  - 第183-185行：`uint? targetColor` → `Color? targetColor`，移除 `.ToUnsignedInteger()` 调用

## API变更记录

| 旧 API | 新 API |
|--------|--------|
| `[SaveableField(n)]` 用于属性 | `[SaveableProperty(n)]` 用于属性 |
| `new MBRandom()` | MBRandom 是静态类，不可实例化 |
| `WeaponComponent.WeaponClass` | `WeaponComponent.PrimaryWeapon.WeaponClass` |
| `CampaignEvents.OnPlayerInventoryExchangeEvent` | `CampaignEvents.PlayerInventoryExchangeEvent` |
| `SetContourColor(uint?)` | `SetContourColor(Color?)` (推测) |
