# 技能配置界面重新设计 v2 — 左侧树形 + 右侧详情

## 任务目标

完全重新设计 CustomSkillScreen，淘汰旧的三列布局+弹窗模式。

## 新布局：方案C（修订版）

### 左侧：树形角色列表
- 三个分组：队伍成员 / 兵种模板 / 领主NPC
- 兵种模板和领主NPC仅在调试模式显示
- 顶部有搜索框
- 只显示角色名称和等级

### 右侧上：技能槽（8个）
- 主技能 / 副技能 / 被动 / 战技 / 法术1~4
- 点击技能槽 → 右侧原位切换为技能目录网格

### 右侧下：原生技能熟练度
- 18项骑砍2原生技能：单手/双手/长杆/弓/弩/投掷/骑术/跑动/锻造/侦察/医术/统御/交易/管理/工程/魅力/流氓/战术
- 通过 hero.GetSkillValue(DefaultSkills.Xxx) 获取

## 交互设计

- 无弹窗 — 右侧面板原位切换
- F12 切换调试模式（解锁兵种模板/领主NPC）
- Tab 循环切换模式
- ↑↓ 导航角色，1~8 选择槽位
- ↑↓←→ 网格导航技能目录，Enter选择，Esc返回

## 技术要点

- 三种目标统一用 CharacterObject.StringId 做 key
- 原生技能熟练度通过 GetSkillValue(SkillObject) 获取
- SkillSelectionPopup 弹窗体系废弃，改为右侧面板显隐切换
